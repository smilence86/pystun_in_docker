# description

Detect nat type with docker command in linux, support amd64, arm32, arm64
  
# usage
```
docker run -it --rm --network=host smilence86/pystun
```

# output
```
NAT Type: Full Cone
External IP: xxx.xxx.xxx.xxx
External Port: 54320
```
  
or
  
```
NAT Type: Blocked
External IP: None
External Port: None
```
